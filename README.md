# Computer Networks II
Undergraduate Subject in AUTH

This project is part of an undergraduate subject in Electrical and Computer Engineering in AUTH.
Subject's name is Computer Networks 2.
Goal of this project is to communicate with the Virtual Lab "Ithaki" through a modem. When connection 
is established, user receives images, soundtracks and is able to control a machine which looks like a 
helicopter, named "Ithaki Copter". Lastly, using the Wireshark programm, we monitor the connection 
between the Virtual Lab and the user.
This assignment is developed in java.
